Last updated: 12 May 2026

technical baseline (~96-97% Seobility, already shipped) to **100% Seobility +

execution of boring tasks for months.

## The ranking gap — why "Webcore Solutions" alone is hard

There are at least 4 competing companies using the name "Webcore" or a near
variant, several of which have been online since 2015 with hundreds of indexed
backlinks and verified Google Business Profiles. Concretely:

- "Webcore" Pakistan-based dev shop (active since ~2015)
- "Webcore Solutions" India IT firm
- "Webcore Inc" US technology consultancy
- "WebCore Solutions" UK web design studio

webcoreuae.com is the newest of these in Google's index. To outrank them on the
plain branded search, Google needs to see consistent signals that webcoreuae.com
is the authoritative answer for "Webcore Solutions in the UAE / Dubai context."

That signal is built through:

1. **Geo-modified content** — pages explicitly targeting "Dubai" / "UAE" / "GCC"
   versions of the query (we now have these — see week-1-on-page.md)
2. **Local citations** — UAE business directory listings with consistent NAP
   (Name, Address, Phone). This is week 3.
3. **Local reviews** — Google Business Profile reviews from real UAE clients.
   This is week 7-8.
4. **UAE-context backlinks** — links from .ae domains and UAE-focused
   publications. This is week 5-6.
5. **Brand mention density** — Google counts how often "Webcore Solutions" is
   mentioned alongside "Dubai" / "UAE" across the web, not just on our site.

These are off-page tasks. Claude cannot create directory listings, sign up for
Google Business, or send pitch emails. **The user must execute these tasks.**

## Commitment required from the user

For this roadmap to work, the user needs to invest **5-10 hours per week** for
the next 12 weeks. Most of that time is:

- Filling out directory profiles (most are 15-30 min each)
- Sending outreach emails (15-20 min per pitch)
- Following up on conversations (5-10 min per check-in)
- Writing blog post drafts (Claude can draft; user reviews/approves)
- Requesting reviews from existing clients (5 min per ask)

If the user cannot commit ~5 hours per week, do not expect this roadmap to
produce results. SEO without consistency is wasted code work.

## A note on honesty

If a week passes and the tasks are not done, do not skip ahead. The plan
assumes earlier weeks compound into later weeks. Skipping week 2 directory
submissions means there are no signals to amplify in week 5 PR outreach.

There is no shortcut. SEO compounds slowly — but it does compound.
