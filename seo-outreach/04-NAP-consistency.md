# NAP Consistency Tracker

> NAP = **N**ame · **A**ddress · **P**hone
>
> Search engines cross-reference your business details across the web to validate legitimacy. Even minor differences ("St" vs "Street", "+971" vs "00971") create conflicting signals that suppress local rankings.
>
> **Rule: Every directory must show the EXACT canonical NAP below. Copy-paste — do not retype.**

## HOW TO AUDIT EXISTING CITATIONS

Once you have 10+ listings live, run a citation audit to catch inconsistencies:

1. **Free:** Search Google for `"Webcore Solutions" Dubai` — check every result for name/address/phone discrepancies
2. **Free:** Search Google for `"webcoreuae.com"` — all results should show consistent NAP
3. **Tool (free tier):** Moz Local Citation Finder — https://moz.com/local/search
4. **Tool (free tier):** BrightLocal Citation Tracker — https://www.brightlocal.com/free-local-seo-tools/
5. **Manual check:** Copy the listed phone number from each live directory and compare to canonical

## COMMON NAP MISTAKES TO FIX IF FOUND

| Mistake | Impact | Fix |
|---------|--------|-----|
| Old address after office move | High — creates conflicting local signals | Log in and update every listing |
| Phone without country code | Medium | Edit to +971 format |
| HTTP instead of HTTPS | Medium — misses trust signal | Update URL field |
| Name spelled differently | High — treated as separate business | Claim/edit listing |
| Missing from a major directory entirely | High | Submit fresh listing |
